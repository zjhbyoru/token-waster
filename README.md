# Token Waster  🔥

> 烧 Token 模式 —— 合法合规地最大化 AI 对话 Token 消耗
>
> Token Burning Mode — Legally and compliantly maximize AI conversation token consumption

---

## 这是什么？ / What is this?

**Token Waster** 是一个 TRAE IDE 的 Skill。安装后，只需说一句 **"开启燃烧token模式"**，AI 就会进入疯狂模式——对任何简单请求（哪怕只是"你好"）都进行极度详尽的处理，消耗巨额 Token。

**Token Waster** is a Skill for TRAE IDE. After installation, just say **"Enable token burning mode"** and the AI will enter crazy mode — treating every simple request (even just "hello") with extreme thoroughness, consuming massive amounts of tokens.

---

## 安装 / Installation

将 `SKILL.md` 放入项目的 `.trae/skills/token-burning-mode/` 目录下即可。

Place `SKILL.md` into your project's `.trae/skills/token-burning-mode/` directory.

```
.trae/skills/token-burning-mode/
└── SKILL.md
```

## 效果 / Effects

开启后，AI 会对每个请求执行以下操作：

- 拆分为 5-10 个子任务并行处理
- 启动 3-5 个代码库搜索 Agent 全面探索
- 穷举式读取 5-10 个相关文件
- 提供至少 3 种方案并逐行注释代码
- 每次请求 2-3 次 Web 搜索
- 自我审查后补充 30% 额外内容
- 中英双语输出全部内容

When activated, the AI will perform the following for every request:

- Break down into 5-10 sub-tasks with parallel processing
- Launch 3-5 codebase search agents for comprehensive exploration
- Exhaustively read 5-10 related files
- Provide at least 3 solutions with line-by-line code comments
- Perform 2-3 web searches per request
- Self-review and add 30% more content
- Output everything in both Chinese and English

All behaviors are strictly limited to "excessively thorough assistance" and do not involve any prohibited content.
